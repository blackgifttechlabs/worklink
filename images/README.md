## Usage in HTML

Replace SVG illustrations with real images like:

  <img src="../images/products/babygrow-yellow.jpg" alt="Fleece Babygrow Yellow" />

or from the homepage:

  <img src="images/categories/toddler-girls-leggings.jpg" alt="Toddler Girls Leggings" />

## Recommended Image Sizes:
- Category circles: 240x240px (will be cropped round)
- Banner images: 1400x500px
- Product cards: 400x530px (portrait)
- Hero background: 1920x600px
